  • A escrita de UIs declarativas no front end foi deprecated. O Beagle é uma plataforma Server Driven UI e todo código de tela deveria estar no Backend. Possibilitar que isso seja feito no front é um anti-pattern além de requerir muita manutenção por parte do time. Ainda é possível escrever telas no front, mas isso será removido na versão 2.0, deixando apenas suporte para JSON (string).
