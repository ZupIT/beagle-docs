## O que é?

O SubmitForm é uma ação **exclusiva** do [**SimpleForm**]({{< ref path="/api/components/forms/simple-form#o-que-é" lang="pt" >}})**,** pois deve ser usada em algum evento dentro do formulário, para que o mesmo seja enviado.
