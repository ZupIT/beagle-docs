## What is it?

Scroll View is a container that shows the preview components by layout scroll.

### ScrollAxis

It is an `ENUM`, the values are:

| Values | Definition |
| :--- | :--- |
| VERTICAL | Defines the scroll as vertical.  |
| HORIZONTAL | Defines the scroll as horizontal.  |
